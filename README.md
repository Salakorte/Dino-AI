# 🦖 Chrome Dino AI (NEAT + Python)

A clone of the Google Chrome Dino game powered by artificial intelligence that learns to play automatically using neuroevolution (NEAT).  
Un clon del juego del dinosaurio de Google Chrome con inteligencia artificial que aprende automáticamente usando neuroevolución (NEAT).

---

## 🚀 Demo / Demo

The AI controls the dinosaur and improves over generations.  
La IA controla el dinosaurio y mejora generación tras generación.

---

## 🧠 How it works / Cómo funciona

This project uses NEAT (NeuroEvolution of Augmenting Topologies), Pygame, and a fitness-based learning system.  
Este proyecto usa NEAT, Pygame y un sistema de aprendizaje basado en fitness.

---

## 🎯 Goal / Objetivo

Train an AI that can learn to jump over obstacles, improve over generations, and survive as long as possible.  
Entrenar una IA capaz de saltar obstáculos, mejorar generación tras generación y sobrevivir el mayor tiempo posible.

---

## 🕹️ Controls / Controles

SPACE → Jump / Saltar

---

## 📦 Installation / Instalación

Clone repository / Clonar repositorio:
git clone https://github.com/yourusername/chrome-dino-ai.git  
cd chrome-dino-ai  

Install dependencies / Instalar dependencias:
pip install pygame neat-python  

---

## ⚠️ Python version / Versión de Python

Recommended: Python 3.10 – 3.12  
Recomendado: Python 3.10 – 3.12  

Python 3.13+ may cause issues / puede dar problemas.

---

## ▶️ How to run / Cómo ejecutar

Train AI / Entrenar IA:
python train.py  

Run AI / Ejecutar IA:
python play.py  

---

## 💾 Model saving / Guardado del modelo

The best model is saved as best_dino.pkl  
El mejor modelo se guarda como best_dino.pkl

Edita config-feedforward.txt para modificar la IA.
